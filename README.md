
# We-Grow-Bank-API
***

### Project Setup
***

### Create environment file.
    -$ touch .env 
### Copy environment configurations to newly created env file
    ~$ cp .env.example .env


### Start scripts.
    ~$ npm install --legacy-peer-deps
    ~$ npm run start:dev [ developement ]
    ~$ npm run start:prod [ production ]

### Using Docker 
    ~$ sudo docker-compose up -d
